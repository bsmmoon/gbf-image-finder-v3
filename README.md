# GBF Image Finder v3

Find images from Granblue Fantasy!

## Installation:
### Install asdf and nodejs
```
git clone https://github.com/asdf-vm/asdf.git ~/.asdf --branch v0.8.0
. $HOME/.asdf/asdf.sh
asdf plugin-add nodejs
asdf install
```

### Install gastby
```
npm install -g gatsby-cli
```

### Install libraries
```
make install
```
